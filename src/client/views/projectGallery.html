Shobu

Shobu is an abstract strategy board game for 2 players. It was designed and developed with my co-designer Manoli Vranas.

As one of the first games we designed together set out to design a classic feeling game akin to chess, to this end we were very succesfull as it garnered amazing feedback on its release and selling out its first print run only a few short months.

Float Escape

Interior Design Project

This project was to design a float therapy center. The aesthetic concept was for a modern clean design with a roman bath inspiration.

The main difficulty was implimenting the design intent is a small space, as well as fitting all the required equipment and office areas in the floor plan.
